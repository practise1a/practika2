Система контроля версий Git

1. Что это?

VCS — инструмент для отслеживания изменений в файлах и управления версиями кода.

2. Git

Git — распределенная система контроля версий, где каждый разработчик имеет полную копию репозитория.

3. Преимущества

  • Возможность востановления данных из определенной версии
  • Возможность ведения командной разработки проекта
  • При поломке сервера система сохраняет данные в локальном репозитории

4. Основные команды

  • git init
  • git add [file]
  • git commit -m "сообщение"
  • git push

5. Git vs GitHub

Git — система контроля версий, GitHub — хостинг для репозиториев Git.
